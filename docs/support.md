# Candor Support

Use this as the support page content for the website and Microsoft Store listing.

## Contact

For alpha support, contact the maintainer who provided your build. Add a public support email here before a Microsoft Store submission.

## Before You Contact Support

Include:

- Candor version.
- Windows version.
- Whether the issue happened during install, recording, transcription, calendar sync, or note saving.
- Any error message shown in the app.

## Common Issues

### Recording does not start

Check that your microphone is connected and that Windows allows desktop apps to use the microphone.

### Transcription is slow

Use the Base or Tiny model in Settings. Larger models are more accurate but need more CPU and memory.

### Calendar events do not appear

Open Settings, disconnect the calendar, then connect it again. If your computer is managed by an employer, calendar or audio permissions may be restricted by IT policy.

### Where are my notes?

Open Settings, then Storage. Candor shows the active local folder used for transcripts and notes.
